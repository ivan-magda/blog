# ivanmagda.dev

Personal blog built with [Astro](https://astro.build/) + [AstroPaper](https://github.com/satnaing/astro-paper) theme, deployed to [GitHub Pages](https://pages.github.com/).

Live at **[ivanmagda.dev](https://ivanmagda.dev)**

## Content

The anchor content is a 9-part series on building a coding agent in Swift — [swift-claude-code](https://github.com/ivan-magda/swift-claude-code).

## Development

```bash
pnpm install          # install dependencies
pnpm run dev          # dev server at localhost:4321
pnpm run build        # production build
pnpm run preview      # preview build locally
pnpm run format       # format with Prettier
pnpm run lint         # lint with ESLint
```
